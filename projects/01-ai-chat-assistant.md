# 项目 1：AI 聊天助手

## 项目目标

跑通 AI 应用最小闭环：

```text
Vue3 -> FastAPI -> 大模型 API -> Vue3
```

## 用户故事

作为用户，我希望在页面输入问题，并看到 AI 回复。

## 功能清单

- [ ] 输入问题
- [ ] 发送问题
- [ ] 显示用户消息
- [ ] 显示 AI 回复
- [ ] loading 状态
- [ ] 错误提示
- [ ] 流式输出
- [ ] 聊天记录
- [ ] 清空会话

## 页面设计

页面区域：

- 左侧：会话列表
- 中间：聊天消息
- 底部：输入框和发送按钮

## 后端接口

```text
POST /api/chat
```

请求：

```json
{
  "message": "什么是大模型？"
}
```

返回：

```json
{
  "reply": "大模型是..."
}
```

## 验收标准

- [ ] 前端能请求后端
- [ ] 后端能调用模型
- [ ] API Key 没有出现在前端
- [ ] 用户能看到 AI 回复
- [ ] 错误时有提示
