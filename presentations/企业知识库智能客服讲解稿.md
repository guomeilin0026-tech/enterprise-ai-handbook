# 企业知识库智能客服讲解稿

## 30 秒介绍

这个项目是一个企业知识库智能客服系统，整合了登录、知识库管理、文档上传、RAG 问答、Agent 工具模拟、用量统计和 Docker 部署，模拟企业 AI 应用从功能到工程化的完整闭环。

## 项目解决什么问题

它解决的是：

```text
企业内部知识分散、重复问答成本高、AI 应用缺少权限和工程化的问题。
```

## 技术栈

```text
前端：Vue3 + Vite
后端：FastAPI
存储：教学版本地 JSON
检索：教学版本地向量检索
部署：Docker Compose
```

## 核心模块

- 用户登录
- 知识库管理
- 文档上传
- RAG 问答
- 引用来源
- Agent 工具调用
- Token 用量统计
- 工具调用日志

## 核心流程

```text
用户登录
-> 创建知识库
-> 上传企业文档
-> 文档切分和索引
-> 用户提问
-> 检索相关片段
-> AI 客服回答
-> 展示引用来源
-> 记录用量
```

## 面试讲法

```text
这个项目是企业 AI 应用的综合实践。我把 RAG 知识库、Agent 工具调用、用户登录、用量统计和 Docker 部署整合到一个系统里。虽然教学版使用本地 JSON 和本地检索，但模块边界按照企业架构设计，后续可以平滑替换为 PostgreSQL、Qdrant 和真实大模型 API。
```

## 可改进点

- PostgreSQL 替换本地 JSON
- Qdrant 替换本地向量检索
- JWT 正式鉴权
- Redis 队列处理文档解析
- Nginx 生产部署
- 多租户权限隔离
