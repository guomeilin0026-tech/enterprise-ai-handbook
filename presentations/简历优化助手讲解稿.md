# 简历优化助手讲解稿

## 30 秒介绍

这个项目是一个 AI 简历优化助手，用户输入简历和岗位 JD，系统通过 Prompt 让大模型输出结构化 JSON，后端用 Pydantic 校验，前端用卡片展示匹配度、优势、不足和修改建议。

## 项目解决什么问题

它解决的是：

```text
如何让 AI 从聊天变成稳定的功能模块。
```

普通 AI 输出一大段文字，前端不好处理。这个项目要求 AI 输出固定 JSON，适合前端渲染和后端校验。

## 技术栈

```text
前端：Vue3 + Vite
后端：FastAPI + Pydantic
模型：OpenAI-compatible API / Mock LLM
```

## 核心流程

```text
输入简历和 JD
-> 后端拼接 Prompt
-> 模型输出 JSON
-> Pydantic 校验
-> 前端卡片展示
```

## 面试讲法

```text
这个项目体现了 Prompt Engineering 和 Structured Output。相比普通聊天，我通过 Prompt 约束模型输出 JSON，并在后端使用 Pydantic 校验，避免模型输出不稳定导致前端渲染失败。
```

## 可改进点

- 增加岗位关键词提取
