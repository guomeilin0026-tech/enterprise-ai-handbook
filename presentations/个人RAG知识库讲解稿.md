# 个人 RAG 知识库讲解稿

## 30 秒介绍

这个项目是一个个人知识库问答系统，用户上传 txt 或 markdown 文档，系统会解析文档、切分文本、建立本地向量索引，用户提问时先检索相关片段，再基于资料生成回答并展示引用来源。

## 项目解决什么问题

它解决的是：

```text
大模型不知道用户私有资料的问题。
```

通过 RAG，系统不是让模型凭空回答，而是先查资料，再让 AI 根据资料回答。

## 技术栈

```text
前端：Vue3 + Vite
后端：FastAPI
检索：教学版本地向量检索
存储：本地 JSON
```

## 核心流程

```text
上传文档
-> 文档解析
-> 文本切分
-> 文本向量化
-> 用户提问
-> 检索相关片段
-> 返回答案和引用来源
```

## 面试讲法

```text
这个项目展示了 RAG 的完整链路。文档上传后会被切分成片段，每个片段建立向量表示。用户提问时，系统先检索最相关片段，再基于这些片段回答，并展示引用来源来降低幻觉风险。
```
