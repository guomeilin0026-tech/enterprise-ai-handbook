# 学习者分层标准

## Level 0：完全小白

特征：

- 不懂后端
- 不懂大模型 API
- 只知道 ChatGPT

目标：

- 能运行 Python
- 能理解 API Key
- 能写最小聊天 Demo

## Level 1：AI 应用入门

特征：

- 会调用大模型 API
- 会写简单接口

目标：

- 能做 AI 聊天助手
- 能做 Prompt 模板
- 能处理基础错误

## Level 2：AI 应用开发者

特征：

- 会 Prompt
- 会结构化输出
- 会基础前后端联调

目标：

- 能做简历优化助手
- 能让 AI 输出 JSON
- 能前端卡片展示

## Level 3：RAG 工程师

特征：

- 会文档解析
- 会 Embedding
- 会向量数据库

目标：

- 能做个人知识库
- 能展示引用来源
- 能减少幻觉

## Level 4：企业 AI 全栈工程师

特征：

- 会 RAG
- 会 Agent
- 会权限和部署

目标：

- 能做企业知识库智能客服
- 能部署上线
- 能讲清楚架构、安全、成本
