# 课程验收标准

一个学习者完成课程后，必须达到以下标准。

## 概念能力

- [ ] 能解释 LLM、Prompt、Token、Embedding、RAG、Agent、MCP
- [ ] 能区分 RAG、Agent、Workflow
- [ ] 能解释企业 AI 应用和模型训练的区别

## 编码能力

- [ ] 能写 FastAPI 接口
- [ ] 能让 Vue3 请求后端
- [ ] 能调用大模型 API
- [ ] 能处理流式输出
- [ ] 能校验结构化输出

## RAG 能力

- [ ] 能上传文档
- [ ] 能解析文档
- [ ] 能切分文本
- [ ] 能生成 Embedding
- [ ] 能使用向量数据库
- [ ] 能做知识库问答
- [ ] 能展示引用来源

## Agent 能力

- [ ] 能设计工具函数
- [ ] 能设计工具参数
- [ ] 能执行工具调用
- [ ] 能记录工具日志
- [ ] 能说明高风险工具为什么要人工确认

## 企业工程能力

- [ ] 能做登录鉴权
- [ ] 能做用户数据隔离
- [ ] 能统计 Token 用量
- [ ] 能写 Docker 部署配置
- [ ] 能说明 API Key 安全策略

## 表达能力

- [ ] 能讲 3 分钟项目介绍
- [ ] 能画系统架构图
- [ ] 能回答 RAG 面试题
- [ ] 能回答 Agent 面试题
- [ ] 能回答企业安全和成本控制问题
