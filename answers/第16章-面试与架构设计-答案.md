# 第 16 章：面试与架构设计参考答案

## 3 分钟项目介绍模板

```text
我做的是一个企业知识库智能客服系统，前端使用 Vue3，后端使用 FastAPI，系统支持用户登录、知识库管理、文档上传、文本切分、RAG 问答、引用来源、Agent 工具调用、用量统计和 Docker 部署。

用户上传文档后，后端会解析并切分文档，建立检索索引。用户提问时，系统先检索相关片段，再基于资料生成回答并展示引用来源。为了贴近企业场景，我还加入了用户权限、工具调用日志和 Token 用量统计。
```

## 如何减少模型幻觉

参考答案：

```text
通过 RAG 检索真实资料、Prompt 限制只根据资料回答、资料不足时拒答、展示引用来源、设置相似度阈值和关键场景人工审核来减少幻觉。
```
