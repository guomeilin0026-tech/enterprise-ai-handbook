# 第 2 章：大模型基础参考答案

## 1. 用自己的话解释 Token

参考答案：

```text
Token 是大模型处理文字时的计量单位。用户输入、系统提示词、历史聊天记录、模型输出都会消耗 Token。Token 越多，费用越高，上下文占用越多，响应也可能越慢。
```

## 2. 为什么 API Key 不能放前端

参考答案：

```text
因为前端代码会被浏览器下载，用户可以看到里面的内容。如果把 API Key 放在前端，别人可以拿走 Key 调用模型，造成费用损失和安全风险。所以 API Key 必须放在后端环境变量里。
```

## 3. Vue3 -> FastAPI -> 大模型 API 流程

参考答案：

```text
用户在 Vue3 页面输入问题
-> Vue3 请求 FastAPI 的 /chat 接口
-> FastAPI 从环境变量读取 API Key
-> FastAPI 调用大模型 API
-> 大模型返回回答
-> FastAPI 把回答返回给 Vue3
-> Vue3 展示 AI 回复
```

## 4. 什么是上下文

参考答案：

```text
上下文是一次请求中模型能看到的全部信息，包括 System Prompt、历史消息、当前问题、RAG 检索资料和工具调用结果。模型并不真正永久记住用户，它是根据每次传入的上下文回答。
```
