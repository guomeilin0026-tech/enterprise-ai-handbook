# 企业 AI 应用全栈学习路线

这份路线告诉你：先学什么、后学什么、每个阶段做到什么程度。

如果你是小白，不要按兴趣乱跳。按阶段走，最稳。

## 总目标

最终你要能独立做出：

```text
企业知识库 + 智能客服系统
```

技术链路：

```text
Vue3 前端
-> Python FastAPI 后端
-> 大模型 API
-> Prompt
-> Structured Output
-> RAG
-> Function Calling
-> Agent
-> Docker 部署
-> 权限和安全
```

## 最推荐的学习顺序

```text
1. AI 行业全景
2. 大模型基础
3. Python + FastAPI 基础
4. AI 聊天助手
5. Prompt
6. Structured Output
7. RAG
8. Function Calling
9. Agent
10. MCP
11. Workflow
12. 多模态
13. 工程化
14. Docker
15. 数据安全
16. 项目案例、产品思维、面试
```

## 不建议一开始学的内容

这些不是不重要，而是现在太早：

- 大模型训练
- Transformer 数学推导
- CUDA
- 分布式训练
- 论文复现

你当前最重要的是：

```text
先做出能跑的企业 AI 应用。
```
