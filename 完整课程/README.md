# 企业 AI 应用全栈课程

这是一套面向前端开发者和 AI 应用初学者的企业 AI 全栈课程。

课程目标：

```text
从会调用大模型 API
到能独立设计、开发、部署企业 AI 应用
```

适合人群：

- 会一点前端，尤其是 Vue3
- 后端基础较弱
- 想学习大模型应用开发
- 想做 RAG、Agent、企业知识库、智能客服等项目

不适合一开始就追求：

- 大模型预训练
- CUDA
- 分布式训练
- 论文复现

## 推荐学习方式

```text
先看一章概念
再做一个小练习
最后把它用到项目里
```

学习顺序建议：

```text
第 1-2 章：建立认知
第 3-5 章：掌握模型输出控制
第 9 章：做企业知识库
第 4、6、7 章：做 Agent 和工具生态
第 8、10 章：扩展业务场景
第 11-13 章：工程化、部署、安全
第 14-16 章：项目、产品、面试
```
