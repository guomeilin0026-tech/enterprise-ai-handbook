# 第 8 章：Workflow（Dify、n8n）

## 8.1 Workflow 是什么

Workflow 是把任务拆成固定流程。

例如：

```text
用户提交工单
-> AI 判断分类
-> 如果是退款，进入退款流程
-> 如果是投诉，进入人工审核
-> 生成处理建议
```

## 8.2 Workflow 和 Agent 的区别

Workflow 更稳定，因为流程是人设计好的。

Agent 更灵活，因为下一步由模型判断。

对比：

```text
Workflow：固定流程，适合标准业务
Agent：动态决策，适合复杂任务
```

## 8.3 Dify 适合什么

Dify 更适合快速搭建：

- 聊天助手
- 知识库问答
- 简单工作流
- 内部 AI 工具原型

## 8.4 n8n 适合什么

n8n 更像自动化工作流平台。

适合：

- 连接多个系统
- 定时任务
- Webhook
- 自动发邮件
- 同步表格和数据库

## 8.5 什么时候自己写代码

需要自己写代码的情况：

- 权限复杂
- 数据安全要求高
- 业务系统深度集成
- 需要自定义前端体验
- 需要高并发和稳定性
