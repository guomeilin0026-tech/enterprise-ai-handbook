# 第 1 章：AI 行业全景（2026）

## 1.1 你要先分清三类 AI 岗位

### 大模型算法工程师

主要做模型训练、微调、推理优化、论文复现、数据处理。

需要能力：

- 深度学习
- Transformer
- PyTorch
- CUDA
- 分布式训练
- 数学和论文能力

这条路更偏算法，不适合作为小白第一起点。

### AI 应用开发工程师

主要做大模型 API 接入、RAG、Agent、企业系统集成。

需要能力：

- 前端页面
- 后端接口
- 数据库
- 大模型 API
- Prompt
- RAG
- Agent
- 部署和安全

这是本课程的主线。

### AI 产品经理

主要判断 AI 能解决什么业务问题，设计产品流程和指标。

需要能力：

- 业务理解
- 用户需求分析
- AI 能力边界
- 成本收益判断
- 产品原型和 PRD

## 1.2 企业为什么需要 AI 应用

企业不是为了炫技而用 AI，而是为了解决具体问题：

- 客服问题重复，想自动回答
- 文档太多，员工找不到信息
- 简历太多，HR 看不过来
- 合同太长，法务审查慢
- 会议太多，需要自动纪要
- 数据报表太多，需要自动总结

这些问题不一定需要训练模型，更多时候需要：

```text
大模型 API + 企业数据 + 业务系统 + 前后端工程
```

## 1.3 企业 AI 应用的主流形态

### AI 聊天助手

最基础形态，用户问，AI 答。

### RAG 知识库

先查企业资料，再让 AI 回答。

适合：

- 公司制度问答
- 产品文档问答
- 客服知识库
- 内部培训资料问答

### Agent 智能体

AI 不只回答，还能调用工具完成任务。

适合：

- 查订单
- 查库存
- 查客户
- 生成报表
- 调用企业内部接口

### Workflow 工作流

把 AI 能力放到固定流程里。

适合：

- 工单分类
- 简历筛选
- 内容审核
- 审批辅助

### 多模态应用

处理图片、PDF、扫描件、视频、音频。

适合：

- OCR 发票识别
- 合同扫描件识别
- 图片问答
- 视频字幕总结
