# 当前版本说明

## 当前定位

当前仓库定位是：

```text
企业 AI 应用全栈工程师从 0 到 1 学习仓库
```

当前项目版本是：

```text
教学版
```

适合用于：

- 学习大模型应用开发
- 理解企业 AI 应用架构
- 跑通 AI 项目完整链路
- 做作品集
- 准备面试
- 作为生产级项目的原型参考

## 当前已经包含

- 16 章课程
- 新手入门指南
- 学习路线
- 每日学习计划
- FAQ
- 故障排查
- 示例代码
- 4 个教学项目
- 工程规范
- 作业答案
- 面试题
- 项目讲解稿
- 架构图
- Docker 项目结构
- GitHub Actions CI

## 当前不是生产级商业系统

当前项目为了降低学习门槛，使用了教学实现：

```text
本地 JSON
mock LLM
简单本地向量检索
模拟 Agent 工具
demo-token
开发模式 Docker
```

这些设计适合学习，但不建议直接用于真实企业生产环境。

## 如果要生产使用，需要升级

生产级建议升级为：

- PostgreSQL 替换本地 JSON
- Qdrant 替换本地向量检索
- 真实 Embedding API
- 真实大模型 API
- JWT 正式鉴权
- Redis 队列
- Nginx 生产部署
- 日志和监控
- 多租户权限隔离
- 文件安全扫描
- Prompt Injection 防护

详细路线：

- [生产级升级路线.md](./生产级升级路线.md)

## 适合学习者的使用方式

建议按这个顺序学习：

```text
新手入门指南
-> 学习路线
-> 完整课程
-> examples

```

## 仓库发布建议

如果公开发布 GitHub，建议在 README 中明确：

```text
这是教学版项目，不是生产级商业系统。
```

这样可以避免学习者误解项目边界。
