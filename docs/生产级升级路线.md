# 生产级升级路线

当前 `apps/` 里的项目是教学版，目标是让学习者先理解完整链路。

如果要升级成企业生产级系统，可以按这份路线推进。

## 1. 存储升级

### 当前教学版

```text
本地 JSON
```

### 生产级方案

```text
PostgreSQL
```

需要保存：

- 用户
- 知识库
- 文档
- 文档片段
- 聊天记录
- Token 用量
- 工具调用日志
- 用户反馈

## 2. 向量检索升级

### 当前教学版

```text
本地简单向量检索
```

### 生产级方案

```text
Embedding API + Qdrant
```

升级后流程：

```text
文本片段
-> Embedding API
-> Qdrant 存向量和 payload
-> 用户问题 Embedding
-> Qdrant 检索 top_k
```

payload 至少包含：

```json
{
  "tenant_id": "企业 ID",
  "user_id": "用户 ID",
  "knowledge_base_id": "知识库 ID",
  "document_id": "文档 ID",
  "chunk_id": "片段 ID",
  "content": "片段内容"
}
```

## 3. 大模型调用升级

### 当前教学版

```text
mock LLM
```

### 生产级方案

```text
OpenAI-compatible API
```

需要支持：

- 模型配置
- temperature 配置
- 流式输出
- 超时处理
- 失败重试
- Token 用量统计
- 多模型切换

## 4. 鉴权升级

### 当前教学版

```text
demo-token
```

### 生产级方案

```text
JWT / 企业 SSO
```

需要支持：

- 登录
- token 过期
- 刷新 token
- 用户角色
- 管理员权限
- 多租户隔离

## 5. 文档处理升级

### 当前教学版

```text
txt / markdown
```

### 生产级方案

支持：

- PDF
- Word
- Excel
- 网页
- 扫描件 OCR

文档处理应该异步化：

```text
上传文档
-> 创建解析任务
-> 后台解析
-> 后台切分
-> 后台向量化
-> 前端查看任务状态
```

## 6. 队列升级

### 当前教学版

```text
接口同步处理
```

### 生产级方案

```text
Redis + Celery / RQ / Arq
```

适合放入队列的任务：

- PDF 解析
- OCR
- 文档切分
- Embedding 批量生成
- 长报告生成

## 7. Agent 升级

### 当前教学版

```text
模拟工具调用
```

### 生产级方案

```text
真实工具注册 + 权限校验 + 工具日志 + 人工确认
```

工具分类：

- 查询类：低风险
- 创建草稿类：中风险
- 删除、付款、发送类：高风险

高风险工具必须人工确认。

## 8. 日志和监控升级

生产级系统需要记录：

- 请求日志
- 模型调用日志
- 工具调用日志
- 文档处理日志
- 错误日志
- Token 用量
- 响应时间

可选工具：

- Prometheus
- Grafana
- Sentry
- OpenTelemetry

## 9. 部署升级

### 当前教学版

```text
Docker Compose 开发模式
```

### 生产级方案

```text
Nginx + Docker Compose / Kubernetes
```

最小生产部署：

```text
Nginx
Vue3 静态文件
FastAPI
PostgreSQL
Qdrant
Redis
对象存储
```

## 10. 安全升级

必须做到：

- API Key 只放后端
- `.env` 不提交 Git
- 用户数据隔离
- 向量检索带权限过滤
- 上传文件限制
- 日志脱敏
- Prompt Injection 防护
- 高风险 Agent 工具人工确认

## 推荐升级顺序

```text
1. PostgreSQL 替换本地 JSON
2. Qdrant 替换本地向量检索
3. 接真实 Embedding API
4. 接真实大模型 API
5. JWT 正式鉴权
6. 文档解析异步任务
7. Redis 队列
8. Nginx 生产部署
9. 日志和监控
10. 多租户权限隔离
```

## 生产级最终架构

```text
Vue3 前端
-> Nginx
-> FastAPI
-> PostgreSQL
-> Qdrant
-> Redis
-> 大模型 API
-> Embedding API
-> 文件存储 / 对象存储
```
