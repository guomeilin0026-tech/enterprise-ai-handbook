# 企业 AI 全栈课程大纲

这份大纲是主线课程目录。

你之前看到的 Python、FastAPI、RAG、Agent、Docker 学习计划，是这套课程里的“代码实操部分”。

这 16 章的定位如下：

```text
第 1-2 章：建立行业和大模型基础认知
第 3-7 章：掌握大模型应用核心能力
第 8-10 章：掌握企业常见 AI 场景
第 11-13 章：掌握工程化、部署和安全
第 14-16 章：项目、产品和面试能力
```

## 第一章：AI 行业全景（2026）

学习目标：

- AI 产品经理是什么

- 企业 AI 项目通常怎么落地

你要能回答：

- 我为什么不先学训练大模型？
- 企业 AI 应用全栈工程师主要做什么？
- Vue3 + Python + 大模型 API 能做哪些项目？

输出成果：

- [ ] 能讲清楚企业 AI 应用开发的完整链路

## 和 16 周实操计划的关系

```text
第 1-2 周：对应第二章大模型基础 + Python/FastAPI 基础
第 3-4 周：对应第二章 + AI 聊天项目
第 5-8 周：对应第三章 Prompt + 第五章 Structured Output
第 9-12 周：对应第九章 RAG
第 13 周：对应第四章 Function Calling + 第七章 Agent
第 14 周：对应第十一章工程化 + 第十三章安全
第 15 周：对应第十四章企业项目案例
第 16 周：对应第十二章部署 + 第十六章面试
```

## 正确学习顺序

你不需要从第一天就学完这 16 章。

推荐顺序：

```text
先学：第二章 + Python/FastAPI 实操
再学：第三章、第五章
然后学：第九章 RAG
再学：第四章、第七章 Agent
最后学：工程化、安全、部署、面试
```

第一章 AI 行业全景可以作为开篇理解，不需要背。

第六章 MCP、第八章 Workflow、第十章多模态属于进阶能力，建议你完成 AI 聊天助手和 RAG 知识库之后再深入。
