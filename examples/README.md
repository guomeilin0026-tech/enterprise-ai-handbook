# 代码示例目录

这个目录用于放每个核心知识点的最小可运行示例。

建议学习方式：

```text
先看完整课程里的概念
再看 examples 里的最小代码
最后把代码合并进 projects 里的实战项目
```

## 示例规划

```text
01-llm-chat/              # 最小大模型聊天
02-prompt-template/       # Prompt 模板
03-structured-output/     # 结构化输出
04-function-calling/      # 工具调用
05-rag-minimal/           # 最小 RAG
06-agent-tools/           # Agent 工具调用
07-docker-compose/        # Docker 部署示例
```

## 安装依赖

在运行 FastAPI 或 Pydantic 示例前，先安装：

```bash
pip install -r requirements.txt
```

也可以在仓库根目录安装：

```bash
pip install -r ../requirements.txt
```
