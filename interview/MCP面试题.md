# MCP 面试题

## 1. MCP 是什么？

MCP 可以理解为 AI 应用连接外部工具和资源的一套标准协议。

## 2. MCP 解决什么问题？

它让 AI 应用可以用更统一的方式发现和使用外部工具、资源和提示词模板，减少每个系统单独适配的成本。

## 3. MCP Client 和 MCP Server 分别是什么？

MCP Client 是使用能力的一方，通常是 AI 应用。MCP Server 是提供工具和资源的一方，例如文件系统、数据库、GitHub 或企业系统。

## 4. MCP 和 Function Calling 有什么区别？

Function Calling 更关注一次模型调用中如何描述和调用函数。MCP 更关注工具和资源如何标准化地提供给 AI 应用。

## 5. MCP 和 Agent 的关系是什么？

Agent 负责判断和决策，MCP 可以为 Agent 提供标准化工具和资源。
