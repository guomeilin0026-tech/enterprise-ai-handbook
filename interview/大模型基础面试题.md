# 大模型基础面试题

## 1. 什么是 Token？

Token 是模型处理文本的计量单位。它会影响上下文长度、调用费用和响应速度。

## 2. 为什么 API Key 不能放前端？

因为前端代码会暴露给用户。如果 API Key 放在前端，别人可以盗用你的额度，造成费用和安全风险。

## 3. 什么是上下文窗口？

上下文窗口是模型一次请求中能看到的最大内容范围，包括系统提示词、历史消息、用户问题、检索资料和工具结果。

## 4. 什么是幻觉？

幻觉是模型编造不存在或不准确的信息。企业应用中常用 RAG、引用来源、结构化校验和拒答策略来减少幻觉。

## 5. 流式输出有什么好处？

流式输出可以让用户更快看到响应过程，提升聊天体验，避免长时间等待完整结果。
