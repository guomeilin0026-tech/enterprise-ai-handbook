# 企业 AI 架构设计题

## 题目 1：设计一个企业知识库智能客服系统

你需要说明：

- 前端使用什么
- 后端使用什么
- 文档如何上传和解析
- 如何做向量化
- 如何检索
- 如何调用大模型
- 如何展示引用来源
- 如何做用户权限
- 如何统计 Token 成本

参考架构：

```text
Vue3 前端
-> FastAPI 后端
-> PostgreSQL 保存用户、文档、聊天记录
-> Qdrant 保存向量
-> 大模型 API 生成回答
-> Docker 部署
```

## 题目 2：如何保证不同企业用户的数据隔离？

关键点：

- 用户、组织、知识库、文档、向量片段都绑定租户 ID
- 检索时必须带租户过滤条件
- 后端统一做权限校验
- 日志中记录访问行为

## 题目 3：如何控制大模型调用成本？

关键点：

- 统计输入和输出 Token
- 限制单次上下文长度
- 限制用户调用次数
- 使用缓存
- 高频简单问题用便宜模型
- 低价值请求拒绝或降级
