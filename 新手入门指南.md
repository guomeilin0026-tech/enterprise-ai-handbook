# 新手入门指南

这份指南给完全不知道怎么开始的人。

如果你是第一次接触企业 AI 应用开发，只需要按这份指南走。

## 你不需要先会什么

你不需要先会：

- 训练大模型
- Transformer 数学
- CUDA
- 分布式训练
- 论文复现

你需要准备：

- 会一点前端更好，不会也可以慢慢补
- 愿意每天写代码
- 愿意按顺序学习，不乱跳

## 这个仓库怎么学

学习顺序：

```text
新手入门指南
-> 学习路线
-> 完整课程
-> examples 示例
-> projects 项目
-> docs 工程规范
-> checklists 验收
-> interview 面试题
```

## 每天怎么学

每天只做 4 件事：

```text
1. 看一个小概念
2. 跟着写一段代码
3. 自己改一个小功能
4. 写 3 句话复盘
```

不要只看文档。必须写代码。

## 你最终会做出什么

最终项目：

```text
企业知识库 + 智能客服系统
```

它包括：

- 用户登录
- 文档上传
- 文档解析
- RAG 知识库问答
- 引用来源
- Agent 工具调用
- Token 用量统计
- Docker 部署

## 第一周你只需要做到什么

第一周不要求你理解所有 AI 概念。

只需要做到：

- 知道企业 AI 应用是什么
- 知道大模型 API 是什么
- 能运行一个 Python 文件
- 能写一个命令行假聊天程序

## 遇到问题怎么办

按这个顺序排查：

1. 重新读当天任务
2. 检查命令是否在正确目录执行
3. 检查依赖是否安装
4. 复制报错信息搜索或提问
5. 不要跳过报错继续往后学

## 学习心态

你不是在背概念。

你是在一步步搭建一个真实企业 AI 系统。

每个概念都要问自己：

```text
它在最终项目里解决什么问题？
```
