# 贡献指南

欢迎一起完善这套企业 AI 应用全栈学习仓库。

## 可以贡献什么

- 修正文档错误
- 补充课程讲解
- 增加代码示例
- 增加项目实战
- 增加面试题
- 增加架构图

## 文档规范

- 面向小白解释，不堆术语。
- 每章尽量包含：概念、流程、案例、练习、常见坑。
- 代码示例优先保持最小可运行。
- 不提交真实 API Key、密码、数据库地址。

## 文件命名

中文文档可以使用中文文件名。

代码示例目录建议使用：

```text
数字-英文描述
```

例如：

```text
01-llm-chat
05-rag-minimal
```

## 提交前检查

- [ ] 没有提交 `.env`
- [ ] 没有提交真实密钥
- [ ] Markdown 链接可访问
- [ ] 示例代码有运行说明
- [ ] 新增内容已在 README 或目录文件中链接
