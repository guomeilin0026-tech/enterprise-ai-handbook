# 完整实战项目源码

这个目录用于放真正可运行的完整项目源码。

学习路径：

```text
examples/ 看最小示例
projects/ 看项目需求
apps/ 跑完整项目源码
```

## 每个项目的标准结构

```text
项目名/
├─ frontend/           # Vue3 前端
├─ backend/            # FastAPI 后端
├─ docker-compose.yml  # 一键启动
├─ .env.example        # 环境变量示例
└─ README.md           # 项目说明
```
